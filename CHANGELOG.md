# Changelog

Todas as mudancas relevantes deste projeto serao documentadas neste arquivo.

## [1.0.0] - 2026-06-11

### Adicionado

- Componente Zeedhi `ZdSignatureInput` para captura de assinatura via canvas.
- Upload opcional de assinatura em `.png`, `.jpg` e `.jpeg`.
- Normalizacao de output para `data:image/png;base64,...`.
- Pacotes separados para model comum, view Vue/Vuetify e agregador Store.
- Demo local com metadata de exemplo.
- Documentacao inicial para uso, eventos, release e submissao para Zeedhi Store.
